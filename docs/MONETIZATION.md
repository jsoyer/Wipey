# Monetization Strategy — Wipey

## Philosophy

Wipey is open source and free at its core. Monetization is based on
**convenience** (App Store), **premium platforms** (iOS, visionOS),
and **community support** (sponsors). No dark patterns, no data selling, ever.

---

## Distribution channels and pricing

| Channel | Price | Audience |
|---|---|---|
| GitHub Releases | Free | Developers, power users |
| Homebrew Cask | Free | macOS power users |
| wipey.app (direct) | Free | General public |
| **Mac App Store** | **$1.99** | Mainstream (convenience) |
| **iOS App Store** | **$0.99** | iPhone / iPad users |
| **visionOS App Store** | **$2.99** | Vision Pro early adopters |
| GitHub Sponsors | Pay what you want | Community supporters |

---

## App Store model — Freemium or One-Time?

**Recommendation: One-Time Purchase (OTP)**

Freemium creates friction for a utility app. Users want it to work immediately.
A single, low price ($1.99) is psychologically easy to accept for a frequently
used tool.

"Pro" features are tied to platforms (iOS, visionOS), not to a paywall
on the open source macOS version.

---

## Features exclusive to paid versions

These features are **not in the open source repo** and are exclusive
to App Store versions:

| Feature | Version | Included in |
|---|---|---|
| Cleaning statistics | macOS App Store | $1.99 |
| Mascot themes / skins | macOS App Store | $1.99 |
| Scheduled cleaning | macOS App Store | $1.99 |
| Full iOS app | iOS App Store | $0.99 |
| Full visionOS app | visionOS App Store | $2.99 |

---

## GitHub Sponsors

Configure GitHub Sponsors on the repo to allow users to support the project.

Suggested tiers:
- **$1/month** — Supporter (badge in README)
- **$5/month** — Fan (name in CONTRIBUTORS)
- **$10/month** — Patron (early beta access)

---

## Ko-fi / Buy Me a Coffee

Link visible on wipey.app and in the app (About screen).
For users who want to make a one-time donation without a subscription.

---

## Conservative projections (6 months post-launch)

| Channel | Users | Conversion | Revenue |
|---|---|---|---|
| GitHub / Homebrew | 5,000 | 0% | $0 |
| Mac App Store | 1,000 | 100% | ~$1,400 (after 30% Apple cut) |
| GitHub Sponsors | 20 | 100% | ~$60/month |
| Ko-fi | 50 | 100% | ~$150 one-time |

> The initial goal is not financial but to build a user base before launching
> iOS and visionOS — the real revenue drivers.
